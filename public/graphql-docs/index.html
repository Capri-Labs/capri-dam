Capri DAM — GraphQL API

Enterprise-grade schema for automated query federation and metadata retrieval. This unified Graph provides zero-noise data fetching for both the React frontend and the isolated AI Gateway.

Authentication

All queries and mutations (except schema introspection in development) require a valid OAuth 2.0 Bearer token obtained from POST /oauth/token.

Queries

Use queries to read assets, collections, and image-processing profiles.

Mutations

Use mutations to write — patch asset metadata, manage collections, and configure image profiles.
