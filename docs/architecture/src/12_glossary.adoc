== Glossary

This section defines the key technical and business terms used throughout the Headless DAM documentation.

[cols="1,3", options="header"]
|===
| Term | Definition
| **Asset** | Any digital file managed by the system, typically a high-resolution "Master" image or video.
| **BYOS (Bring Your Own Storage)** | An architectural pattern where the DAM connects to the customer's existing cloud storage (S3, Azure, GCP) rather than providing its own proprietary storage.
| **Edge Computing** | Processing data (like image transformation) at the "edge" of the network, physically close to the user, typically via a CDN like Fastly.
| **Fastly IO (Image Optimizer)** | The real-time image processing service that performs resizing, cropping, and format conversion on the fly.
| **Focal Point** | X and Y coordinates (normalized 0.0 to 1.0) stored in metadata to indicate the center of interest in an image for "Smart Cropping."
| **Headless** | A software architecture where the backend (content/logic) is decoupled from the frontend (presentation), communicating via APIs.
| **HMAC Signing** | (Hash-based Message Authentication Code). A security mechanism used to sign delivery URLs to ensure that parameters (like width/height) have not been tampered with.
| **Metadata** | Data providing information about one or more aspects of an asset (e.g., file size, dimensions, tags, focal points).
| **Origin Shielding** | A Fastly feature that designates a specific cache node to handle all requests to the storage bucket, significantly reducing egress costs from the cloud provider.
| **Purge (Invalidation)** | The process of removing an asset (or all its renditions) from the CDN cache so that the next request fetches the most recent version.
| **Rendition** | A specific version of a master asset (e.g., a 200x200 thumbnail) generated by the transformation engine.
| **Surrogate Key** | A unique header/tag applied to assets at the edge. It allows for "instant" purging of thousands of related image renditions using a single API call.
| **TCO (Total Cost of Ownership)** | The total financial estimate of the project, including infrastructure, labor, and maintenance.
| **Transformation** | The act of changing an image's properties (dimensions, format, quality) on-the-fly.
|===
