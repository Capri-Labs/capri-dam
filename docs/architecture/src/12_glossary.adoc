[[section-glossary]]
== Glossary

[cols="1,3", options="header"]
|===
| Term | Definition

| **ACL (Access Control List)**
| The 7-column permission matrix on a `FolderPolicy` record:
  _read_, _modify_, _create_, _delete_, _replicate_, _manage_, _explicit_deny_.
  A single `explicit_deny` flag overrides all positive grants for that group.

| **ADR (Architecture Decision Record)**
| A short document capturing an important architectural choice, its context,
  alternatives considered, and the rationale for the final decision.

| **arc42**
| A pragmatic template for software architecture documentation, consisting of
  12 standardised sections. See https://arc42.org.

| **Asset**
| Any digital file managed by the DAM: image (JPG, PNG, TIFF, WebP, AVIF),
  video (MP4, MOV), document (PDF), or raw binary.

| **BYOS (Bring Your Own Storage)**
| A model in which the customer supplies and owns the cloud object storage
  bucket (S3, Azure Blob, or GCP); the DAM only stores references and metadata.

| **CDN (Content Delivery Network)**
| A geographically distributed network of edge servers that cache and serve
  assets to end-users with low latency. In this system, provided by Fastly.

| **Closure Table**
| A database pattern used to represent hierarchical data (e.g. `UserGroup`
  parent-child relationships) by storing every ancestor-descendant pair with
  a distance column. Enables efficient hierarchy traversal without recursive SQL.

| **DAM (Digital Asset Management)**
| A system for organising, storing, and distributing digital content.

| **Devise**
| A flexible Rails authentication gem. Used for local username/password login.
  Keycloak SSO is added via OmniAuth.

| **Explicit Deny**
| An RBAC override flag on `FolderPolicy`. When `true` for any group a user
  belongs to, all access to that folder is denied regardless of other grants.

| **Fastly IO (Image Optimizer)**
| A Fastly CDN add-on that performs real-time image transformations
  (resize, crop, format conversion, WebP auto) at the edge server on first
  request and caches the result globally.

| **FolderPolicy**
| A join-table record linking a `UserGroup` to a `Folder` with the 7-column
  ACL matrix. Policies cascade from parent to child folders by default.

| **GraphQL**
| A query language and runtime for APIs that allows clients to request exactly
  the data they need. Used alongside the REST API v1 in this system.

| **HMAC (Hash-based Message Authentication Code)**
| A cryptographic signature mechanism. Used to sign Fastly delivery URLs so
  that only the DAM API can produce valid image transformation parameters.

| **Hybrid Monolith**
| The architectural pattern used by this project: a single Rails application
  that serves both server-rendered HTML views and JSON APIs. React components
  are mounted into specific DOM nodes rather than running as a separate SPA.

| **Image Profile**
| A named set of Fastly IO transformation parameters (width, height, crop
  mode, output format, quality). Profiles are assigned to folder paths so
  the correct preset is applied automatically.

| **Ingestion Batch**
| A bulk-upload operation submitted to the system. Each batch contains one or
  more `IngestionItem` records representing individual assets to be processed
  by background workers.

| **Kamal**
| A zero-downtime Rails deployment tool (by Basecamp). Manages Docker
  container deployment over SSH to any cloud VM.

| **Metadata Schema**
| A user-defined set of typed fields (text, number, date, enum) that can be
  assigned to a folder path. Assets in that folder must conform to the schema.

| **OIDC / OAuth2**
| Open standards for identity federation. Used to integrate with Keycloak
  (or other IdPs) for single sign-on (SSO) in enterprise deployments.

| **RBAC (Role-Based Access Control)**
| The permission model used by the DAM. Users belong to `UserGroup` s;
  groups are assigned `FolderPolicy` records that define what operations are
  allowed or denied on each folder.

| **Rendition**
| A specific size/format variant of an asset (e.g. 800×600 JPEG WebP).
  This system generates renditions _on-the-fly_ at the Fastly edge rather
  than pre-storing them ("rendition bloat avoidance").

| **Sidekiq**
| A Ruby background job framework backed by Redis. Used for all async work:
  ingest, metadata extraction, AI embedding, email, CDN purge, and reports.

| **Surrogate Key**
| A unique tag applied to all Fastly-cached responses for a given asset.
  Sending a "Purge by Surrogate Key" request to the Fastly API instantly
  clears all renditions of that asset from every edge server.

| **Typesense**
| An open-source, typo-tolerant search engine used as a sidecar index for
  sub-second full-text search over the asset library.

| **UserGroup**
| A named collection of users, organised in a hierarchy (parent/child) using
  a closure table. Groups are the unit of RBAC assignment in this system.

| **UserGroupClosure**
| The closure-table join record linking `ancestor_id`, `descendant_id`, and
  `distance`. Used to efficiently query all ancestors or descendants of a group.

| **VCL (Varnish Configuration Language)**
| The scripting language used to configure Fastly edge behaviour (caching
  rules, request/response manipulation, auth checks).

| **Video Profile**
| A named set of video encoding parameters (codec, resolution, bitrate,
  container format) assigned to folder paths for the transcoding pipeline.
|===
