This guide is the authoritative reference for contributors and integrators working on
**Capri DAM**.  It covers every layer of the stack — data models, REST and GraphQL APIs,
React components, authentication and RBAC, storage backends, background workers,
the workflow engine, semantic search, and the test suite.
