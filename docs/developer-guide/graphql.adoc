[[section-graphql]]
= GraphQL API

The GraphQL endpoint is at `POST /graphql`.
SDL schema: `swagger/graphql/schema.graphql`.
Interactive Playground: http://localhost:3000/api/graphql (SpectaQL HTML docs).

Regenerate after changes: `make graphql-docs`.

== Authentication

Same as REST — send `Authorization: Bearer <token>` in the HTTP header.
Session cookies also work for the browser-based playground.

== Depth & complexity limits

[source,ruby]
----
# app/graphql/headless_dam_schema.rb
max_depth      10
max_complexity 300
----

== Queries

=== `asset(id: ID!): Asset`

Fetch a single asset with all fields.

[source,graphql]
----
query {
  asset(id: "42") {
    id title contentType fileSize
    metadata customMetadata
    focalPointX focalPointY
    folder { id name path }
    versions { id createdAt }
    collections { id name }
  }
}
----

=== `assets(folderId, q, page, perPage, status): AssetConnection`

Paginated asset list with optional full-text search.

[source,graphql]
----
query {
  assets(folderId: "5", q: "hero banner", perPage: 10) {
    nodes { id title contentType }
    pageInfo { hasNextPage endCursor }
  }
}
----

=== `folder(id: ID!): Folder`

Returns folder with `children`, `assets`, and `activePolicies`.

=== `folders(parentId): [Folder!]`

List root or child folders.

=== `search(q, folderId, contentTypes, semantic): SearchResult`

Unified search returning assets and folders.

[source,graphql]
----
query {
  search(q: "product launch", semantic: true) {
    assets { id title score }
    total
  }
}
----

=== `collections(page, perPage): CollectionConnection`

Paginated collection list.

=== `collection(id: ID!): Collection`

=== `me: User`

Returns the authenticated user with group memberships and preferences.

=== `notifications(read): [Notification!]`

=== `workflowTasks(status): [WorkflowTask!]`

=== `binAssets(page, perPage): AssetConnection`

Trashed assets accessible to the caller.

=== `duplicateGroups(page, perPage): DuplicateGroupConnection`

=== `metadataSchemas: [MetadataSchema!]`

=== `imageProfiles: [ImageProfile!]` / `videoProfiles: [VideoProfile!]`

=== `systemStatus: SystemStatus`  _(admin only)_

Returns queue depths, storage usage, and health indicators.

== Mutations

=== Asset mutations

[cols="1,3", options="header"]
|===
| Mutation | Description

| `createAsset(input)` | Upload an asset. `input` includes `folderId`, `title`, `file` (base64 or presigned flow).
| `updateAsset(id, input)` | Update metadata: `title`, `description`, `customMetadata`, `focalPointX/Y`.
| `deleteAsset(id)` | Soft-delete (move to bin).
| `restoreAsset(id)` | Restore from bin.
| `permanentlyDeleteAsset(id)` | Hard-delete (admin/manage permission required).
| `bulkRestoreFromBin(ids)` | Restore multiple assets.
| `emptyBin` | Hard-delete all trashed assets the caller can manage.
| `updateBinRetentionPolicy(input)` | Set `retentionDays`, `workflowBehavior`.
| `triggerBinPurge` | Force run `BinPurgeWorker` (admin only).
|===

=== Folder mutations

[cols="1,3", options="header"]
|===
| Mutation | Description

| `createFolder(input)` | Create folder: `name`, `parentId`.
| `updateFolder(id, input)` | Rename or redescribe.
| `deleteFolder(id)` | Soft-delete with descendant cascade.
|===

=== Collection mutations

[cols="1,3", options="header"]
|===
| Mutation | Description

| `createCollection(input)` | Create manual or smart collection.
| `updateCollection(id, input)` | Rename, change rules.
| `deleteCollection(id)` | Remove collection (assets unaffected).
| `addAssetToCollection(collectionId, assetId)` | Add to manual collection.
| `removeAssetFromCollection(collectionId, assetId)` | Remove from manual collection.
|===

=== Duplicate mutations

[cols="1,3", options="header"]
|===
| Mutation | Description

| `resolveDuplicateGroup(id, keepId)` | Keep one asset, trash the rest in the group.
|===

=== Notification mutations

[cols="1,3", options="header"]
|===
| Mutation | Description

| `markNotificationRead(id)` | Mark a single notification as read.
| `markAllNotificationsRead` | Mark all.
|===

=== User & Auth mutations

[cols="1,3", options="header"]
|===
| Mutation | Description

| `updateUserPreferences(input)` | Update locale, theme, notification flags.
| `createPersonalAccessToken(name, expiresAt)` | Returns the plain-text token (shown once).
| `revokePersonalAccessToken(id)` | Hard-delete the token.
|===

== Type reference

=== Asset

[source,graphql]
----
type Asset {
  id: ID!
  title: String!
  description: String
  contentType: String!
  fileSize: Int!
  checksum: String!
  status: AssetStatus!
  metadata: JSON
  customMetadata: JSON
  focalPointX: Float
  focalPointY: Float
  folder: Folder!
  createdBy: User!
  createdAt: ISO8601DateTime!
  updatedAt: ISO8601DateTime!
  versions: [AssetVersion!]!
  collections: [Collection!]!
  workflowInstances: [WorkflowInstance!]!
  deliveryUrl(width: Int, height: Int, format: String, profile: String): String
}
----

=== Folder

[source,graphql]
----
type Folder {
  id: ID!
  name: String!
  slug: String!
  path: String!
  parent: Folder
  children: [Folder!]!
  assetsCount: Int!
  activePolicies: [FolderPolicy!]!
  metadataSchema: MetadataSchema
  imageProfile: ImageProfile
}
----

=== User

[source,graphql]
----
type User {
  id: ID!
  email: String!
  role: UserRole!
  groups: [UserGroup!]!
  preferences: UserPreference!
}
----

=== WorkflowTask

[source,graphql]
----
type WorkflowTask {
  id: ID!
  workflow: Workflow!
  asset: Asset
  assignee: User!
  status: TaskStatus!
  decision: String
  comment: String
  dueAt: ISO8601DateTime
}
----
