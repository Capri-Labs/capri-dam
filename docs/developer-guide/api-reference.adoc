[[section-api-reference]]
= REST API Reference

All REST endpoints live under `/api/v1/` and are documented with **rswag** (OpenAPI 3.0).
Regenerate after changes: `make swagger-docs`.
Interactive docs: http://localhost:3000/api/rest

== Authentication

Every request must include one of:

[cols="1,3", options="header"]
|===
| Method | Header / Param

| **Bearer token** (Doorkeeper OAuth2) | `Authorization: Bearer <token>`
| **Personal Access Token** | `Authorization: Bearer <pat_token>`
| **Session cookie** | Set by Devise login (`POST /users/sign_in`).
|===

== Response envelope

[source,json]
----
{
  "data": { ... },       // present on success
  "meta": { "total": 120, "page": 1, "per_page": 25 },
  "errors": [ { "code": "not_found", "detail": "Asset not found" } ]
}
----

HTTP status codes follow RFC 7231: `200 OK`, `201 Created`, `204 No Content`,
`400 Bad Request`, `401 Unauthorized`, `403 Forbidden`, `404 Not Found`,
`422 Unprocessable Entity`.

== Assets

=== List assets
`GET /api/v1/assets`

[cols="1,1,3", options="header"]
|===
| Param | Type | Description

| `folder_id` | integer | Filter by folder.
| `q` | string | Full-text keyword search.
| `content_type` | string | MIME type prefix (`image/`, `video/`, ‚Ä¶).
| `status` | string | `active` (default), `archived`, `all`.
| `page` / `per_page` | integer | Pagination (default: 25, max: 200).
| `sort` | string | `created_at`, `updated_at`, `title`, `file_size`.
| `dir` | string | `asc` / `desc` (default: `desc`).
|===

=== Create / upload asset
`POST /api/v1/assets`

Multipart form data:

[cols="1,1,3", options="header"]
|===
| Field | Required | Description

| `file` | ‚úì | Binary payload.
| `folder_id` | ‚úì | Target folder ID.
| `title` | | Overrides filename.
| `description` | | Free text.
| `custom_metadata` | | JSON string matching the folder's `MetadataSchema`.
|===

=== Get asset
`GET /api/v1/assets/:id`

Returns the full asset record including metadata, storage URL, versions, and
computed delivery URLs.

=== Update asset metadata
`PATCH /api/v1/assets/:id`

Permitted params: `title`, `description`, `custom_metadata`, `focal_point_x`,
`focal_point_y`, `status`.

=== Delete (trash) asset
`DELETE /api/v1/assets/:id`

Soft-deletes the asset (moves to Recycle Bin).  See <<section-bin>>.

=== Upload new version
`POST /api/v1/assets/:id/versions`

Replaces the master file; the previous master is saved as `AssetVersion`.

=== List versions
`GET /api/v1/assets/:id/versions`

=== Asset delivery URL
`GET /api/v1/assets/:id/url`

Returns a signed Fastly IO delivery URL with optional transformation params:
`width`, `height`, `crop`, `format`, `quality`, `profile_id`.

== Folders

=== List folders
`GET /api/v1/folders` ‚Äî returns root-level folders.

`GET /api/v1/folders/:id/children` ‚Äî sub-folders.

=== Create folder
`POST /api/v1/folders`

Params: `name`, `parent_id` (optional), `description`.

=== Get folder
`GET /api/v1/folders/:id`

Includes `path`, `children_count`, `assets_count`, `active_policies`.

=== Update folder
`PATCH /api/v1/folders/:id` ‚Äî `name`, `description`.

=== Delete folder
`DELETE /api/v1/folders/:id` ‚Äî soft-deletes folder and all descendant assets.

== Collections

=== List collections
`GET /api/v1/collections`

=== Create collection
`POST /api/v1/collections`

Params: `name`, `collection_type` (`manual`/`smart`/`temporal`), `rules` (array for smart).

=== Add / remove asset
`POST /api/v1/collections/:id/assets` ‚Äî body: `{ asset_id: 42 }` +
`DELETE /api/v1/collections/:id/assets/:asset_id`

== Search

=== Full-text + faceted search
`GET /api/v1/search`

[cols="1,1,3", options="header"]
|===
| Param | Type | Description

| `q` | string | Keyword query (full-text).
| `folder_id` | integer | Scope to folder subtree.
| `content_type[]` | array | MIME type filters.
| `tags[]` | array | Tag filters.
| `date_from` / `date_to` | date | Upload date range.
| `semantic` | boolean | Enable pgvector semantic re-ranking (requires AI config).
| `page` / `per_page` | integer | Pagination.
|===

=== Semantic similarity
`POST /api/v1/search/similar`

Body: `{ asset_id: 42, limit: 10 }` ‚Äî returns assets nearest in embedding space.

== Recycle Bin

=== List trashed assets
`GET /api/v1/bin`

=== Restore single asset
`POST /api/v1/bin/:id/restore`

=== Bulk restore
`POST /api/v1/bin/bulk_restore` ‚Äî body: `{ ids: [1,2,3] }`

=== Empty bin (permanent delete)
`DELETE /api/v1/bin` ‚Äî permanently destroys all trashed assets the caller can manage.

=== Retention policy
`GET /api/v1/bin/retention_policy` +
`PUT /api/v1/bin/retention_policy` ‚Äî body: `{ retention_days: 30, workflow_behavior: "block" }`

=== List schemas
`GET /api/v1/metadata_schemas`

`POST /api/v1/metadata_schemas` ‚Äî body: `{ name, folder_path, fields: [...] }`

== Metadata Export / Import

=== Trigger export
`POST /api/v1/metadata_exports` ‚Äî body: `{ folder_id, format: "csv" }` +
Returns an `IngestionBatch`-style record; download URL available when `status = completed`.

=== List exports
`GET /api/v1/metadata_exports`

=== Upload import file
`POST /api/v1/metadata_imports` ‚Äî multipart: `file` (CSV).

=== Poll import status
`GET /api/v1/metadata_imports/:id`

== Ingestion (Bulk Import)

=== Create batch
`POST /api/v1/ingestion_batches`

Body: `{ source_type, config: { ... } }` where `config` contains source credentials.
Supported `source_type` values: `s3`, `gcs`, `nuxeo`, `brandfolder`, `cloudinary`,
`sharepoint`, `bynder`, `aprimo`, `widen`, `aem`, `ftp`, `mediavalet`, `extensis`, `canto`.

=== Poll batch status
`GET /api/v1/ingestion_batches/:id`

=== List batch items
`GET /api/v1/ingestion_batches/:id/items`

== CDN & Edge

=== CDN configuration
`GET /api/v1/cdn_configurations` +
`POST/PATCH/DELETE /api/v1/cdn_configurations/:id`

`POST /api/v1/edge_operations/purge` ‚Äî body: `{ asset_ids: [...] }` ‚Äî triggers Fastly surrogate-key purge.
`POST /api/v1/edge_operations/purge_all` ‚Äî purge entire cache (admin only).

=== Webhooks
`GET /api/v1/webhooks` +
`POST /api/v1/webhooks` ‚Äî body: `{ url, events: ["asset.created", "asset.deleted"] }`

== AI & Embeddings

`POST/PATCH /api/v1/ai_configurations/:id`

=== AI Copilot (chat)
`POST /api/v1/copilots/chat` ‚Äî body: `{ message, context_asset_id? }` ‚Äî streamed response.

| `GET /admin/report_snapshots/:id` | Download generated report files.
| `GET/POST/PATCH /admin/email_templates` | Transactional email templates.
| `GET /admin/email_deliveries` | Delivery log.
| `POST /admin/migrations` | Trigger schema/data migration jobs.
| `GET/POST/PATCH /admin/folder_policies` | Folder-level ACL management.

|===
