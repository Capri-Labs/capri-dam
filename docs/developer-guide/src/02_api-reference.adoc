[[section-api-reference]]
= REST API Reference

Capri DAM exposes conventional CRUD endpoints plus operational APIs for admin,
AI, ingestion, workflow, and governance modules. The canonical routes live under
`app/controllers/api/v1/` and `app/controllers/admin/`.

== Authentication summary

[cols="1,3", options="header"]
|===
| Mechanism | Notes
| Session cookie | Used by the Rails/React shell for browser traffic.
| OAuth2 bearer token | Backed by Doorkeeper.
| Personal access token | Hashed token stored in `PersonalAccessToken`.
| Gateway shared secret | Machine-to-machine callbacks such as agent execution,
AI batch progress, and health callbacks.
|===

== Collections, bin, duplicates, and delivery

[cols="1,2,1,3", options="header"]
|===
| Method | Path | Auth | Description

| `GET` | `/api/v1/collections` | user | List active collections with optional smart/manual filtering.
| `POST` | `/api/v1/collections` | user | Create a collection.
| `GET` | `/api/v1/collections/:slug` | user | Collection detail, optionally time-travelled with `as_of`.
| `PATCH` | `/api/v1/collections/:slug` | user | Update collection metadata.
| `DELETE` | `/api/v1/collections/:slug` | user | Archive/soft-delete a collection.
| `POST` | `/api/v1/collections/:slug/configure_rule` | user | Upsert the semantic `CollectionRule`.
| `POST` | `/api/v1/collections/:slug/simulate_rule` | user | Dry-run semantic routing.
| `GET` | `/api/v1/collections/:slug/cluster_map` | user | 2D semantic cluster-map nodes.
| `POST` | `/api/v1/collections/:slug/assets` | user | Add asset to manual collection.
| `DELETE` | `/api/v1/collections/:slug/assets/:asset_id` | user | Remove membership.
| `POST` | `/api/v1/collections/:slug/assets/:asset_id/toggle_pin` | user | Toggle pinned state on `CollectionAsset`.
| `PATCH/DELETE` | `/api/v1/collections/bulk_update`, `/api/v1/collections/bulk_delete` | user | Bulk workspace operations.
| `GET/PATCH` | `/api/v1/collection_settings` | user | Collection module defaults.
| `GET` | `/api/v1/bin` | user | Paginated recycle-bin contents.
| `GET` | `/api/v1/bin/stats` | user | Aggregate trash metrics.
| `POST` | `/api/v1/bin/bulk_restore` | user | Restore mixed asset/folder selections.
| `DELETE` | `/api/v1/bin/bulk_destroy` | user | Permanently purge selected trashed items.
| `DELETE` | `/api/v1/bin/empty` | user | Empty the entire recycle bin.
| `GET/PATCH` | `/api/v1/bin/retention_policy` | admin | Read/update purge retention settings.
| `POST` | `/api/v1/bin/trigger_purge` | admin | Enqueue a purge cycle.
| `GET` | `/api/v1/bin/purge_status` | admin | Read current purge-job lock/status markers.
| `GET` | `/api/v1/duplicate_groups` | user | List duplicate groups by status.
| `GET` | `/api/v1/duplicate_groups/stats` | user | Duplicate counts by lifecycle state.
| `GET` | `/api/v1/duplicate_groups/:id` | user | Group detail with member assets.
| `PATCH` | `/api/v1/duplicate_groups/:id/resolve` | user | Resolve by keeping all or deleting duplicates.
| `PATCH` | `/api/v1/duplicate_groups/:id/dismiss` | user | Mark false positive / defer action.
| `PATCH` | `/api/v1/duplicate_groups/bulk_resolve` | user | Bulk duplicate resolution.
| `GET/PATCH` | `/api/v1/cdn_configurations` | admin | Read/update CDN provider settings.
| `POST` | `/api/v1/edge_operations/purge` | admin | Purge CDN cache or surrogate keys.
| `POST` | `/api/v1/edge_operations/sign_url` | admin | Return a signed delivery URL.
| `POST` | `/api/v1/edge_operations/sync_metadata` | admin | Push edge metadata refresh.
