This guide collects **operational, step-by-step deployment playbooks** for running Capri DAM in
production. Unlike the <<section-deployment-view,architecture deployment view>> (which explains
*why* the system is shaped the way it is), this guide is a hands-on runbook: exact commands,
exact files, exact troubleshooting steps.

Each deployment target gets its own numbered chapter so this guide can grow over time (e.g. a
future chapter for Kubernetes/Helm, or a managed-PaaS target) without disturbing existing content.
The first supported target is **Kamal 2** — Docker-based deployment to plain Linux servers (or a
single VPS), using `kamal-proxy` for zero-downtime rollouts and automatic Let's Encrypt TLS.
