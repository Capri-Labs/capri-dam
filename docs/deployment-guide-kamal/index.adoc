This guide collects **operational, step-by-step deployment playbooks** for running Capri DAM in
production using **Kamal 2**. Unlike the <<section-deployment-view,architecture deployment view>>
(which explains *why* the system is shaped the way it is), this guide is a hands-on runbook: exact
commands, exact files, exact troubleshooting steps.

Two other deployment targets are documented as sibling guides — see
link:../deployment-guide-aws/[AWS (ECS Fargate)] and
link:../deployment-guide-azure-kubernetes/[Azure Kubernetes Service (AKS)] — for teams that prefer
a managed cloud platform over Kamal's plain-Linux-server model.
